Paste to Markdown

Convert rich clipboard content to Markdown locally, then edit or preview the result in one focused workspace.

How to use

Paste from the source you already use. The conversion happens on this device.

  1. Copy rich content with {copyShortcut}
  2. Paste on this page with {pasteShortcut}
  3. Edit, preview, then copy the Markdown result
  4. Press Escape to reset
  5. Switch tabs with {editShortcut} and {previewShortcut}

This converter is based on to-markdown. All data is processed locally, nothing is sent to any server.

Source code available at GitHub, Issues.
